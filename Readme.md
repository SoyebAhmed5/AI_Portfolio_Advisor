# 🚀 AI Portfolio Advisor

An AI-powered career guidance platform that analyzes a user's **LinkedIn profile** and **Resume (PDF)** to provide personalized career recommendations, skill assessments, learning roadmaps, and actionable insights using the **OpenAI API**.

## 📌 Overview

AI Portfolio Advisor helps students, job seekers, and professionals understand their current profile, identify skill gaps, and receive AI-generated recommendations to accelerate their career growth.

The platform generates detailed career advice, recommends learning resources, creates a personalized roadmap, and provides visual insights in an easy-to-understand format.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SoyebAhmed5/AI_Portfolio_Advisor.git
```

```bash
cd AI_Portfolio_Advisor
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_api_key
PORT=3000
```

### Start Development Server

```bash
npm run dev
```

### Start Production Server

```bash
npm start
```

## 🚀 How It Works

1. Upload Resume PDF or provide LinkedIn profile information.
2. Portfolio data is extracted and processed.
3. OpenAI analyzes the profile.
4. AI generates:

   * Career Advice
   * Skill Assessment
   * Career Roadmap
   * Course Recommendations
5. Results are displayed through visual dashboards.
6. Users can export the generated report as a PDF.
